Progama de la Especialidad de Programacion CBTIS No.16

Perfil de egreso

La formacion que ofrece la carrera Tecnica en Programacion permite el egresado, a traves de la articulacion de saberes de diversos campos, realizar actividades dirigidas a la:


  • Instalacion y desrrollo de software de aplicacion utilizando programacion estructurada y orientada a a objetos de abientes web y moviles, con almacenamiento persistente de datos,
  • Asi como la configuracion y administracion de plataforma e-learning y
  • Comercio electronico. Durante el proceso de formacion de los cinco modulos, el estudiante desarrolla o reforzara:
    Las siguientes competencias profesionles:
    • Desarrolla e instala software de aplicacion utilzando programacion estructurada, con al amacenamiento persistente de los datos
    • Desarrolla software de aplicacion utilizando programacion orientada a objetos, con alamacenamientos persistente de los datos
    • Desarrola aplicaciones web y moviles
    • Administra sistemas operativos, aplicaciones y servicios
    • Administra sistemas operativos,aplicaciones y servicios
    • Desarrolla,admistra y configura soluciones de e-learning y comercio electronico

    • Modulo I

      Desarrollo e instala software de aplicacion utilizando programacion estructurada,con alamacenamiento de los datos

    • Modulo II

      Desarrolla software de aplicacion utilizando programacion orientada a objetos, con almacenamiento persistente de los datos

    • Modulo III

      Desarrolla aplicaciones web y moviles

    • Modulo IV

      Administra sistemas operativos,aplicaciones y servicios

    • Modulo V

      Desarrolla,admistra y configura soluciones de e-learning y comercio electronico
